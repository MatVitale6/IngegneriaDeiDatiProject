Bisogna creare uno schema mediato per integrare tutte le sorgenti di dati riguardanti delle aziende.
Lo schema mediato deve allineare tutti gli schemi delle sorgenti e deve avere almeno 20 attributi. 
Per esempio un attributo 'Name' ed un altro attributo 'name' possono essere allineati sotto lo stesso attributo 'name'. (Usa anche delle semplici tecniche di normalizzazione del testo per fare match).
Il task difficile si prensenta nel momento in cui due attributi grammaticalmente diversi possono riferirsi allo stesso concetto, in questo caso è necessaria la tua comprensione semantica. Per riuscire in quest'ultimo puoi anche valutare il significato delle 5 righe campionate che ho messo per esempio nel dataset fornito qui sotto.
Il JSON fornito in formato testuale contiene dati strutturati in questa maniera.

Proponi uno schema mediato che unifichi i dati. Lo schema dovrebbe avere campi normalizzati, tipi di dati coerenti ed una struttura che minimizzi la perdita di informazioni.
Produci del testo in formato JSON (non file da scaricare) per l'output.

Commenta anche gli attributi che hai unito e da quale sorgente sono stati ereditati.
